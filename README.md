# work-notes
作業メモプラグイン
（このリポジトリは作業メモのためのWordPressプラグインです。）

---

## 📌 特徴
- 主な機能や特徴を箇条書きでまとめます
- 例：〇の自動化
- 例：△形式での出力

---

## 🚀 インストール方法
- wp-content/plugins/work-notes/ に配置
- 管理画面で有効化

---

## 💻 使い方
- 「設定＞作業メモ」で依頼元/担当者を登録
- 投稿編集画面のメタボックス説明
- 「作業一覧」サブメニューの説明

---

- PHP 7.4+（推奨8.1+）
